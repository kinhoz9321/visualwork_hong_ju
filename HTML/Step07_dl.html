Definition list (정의형 목록) dl

HTML
Hyper Text Markup Language의 약자이다.
CSS
design적인 요소를 결정한다.
Javascript
program language적인 요소, 웹브라우저의 동작을 담당한다.

dl, dt, dd 를 합쳐서 정의형 목록이라고 한다.
dl 이라는 요소가 있고, 자식 요소로 dt, dd 가 있다.
dd 는 디폴트로 왼쪽 마진을 가지고 있다.

dl definition list 정의 목록 용어를 설명하는 목록
dt definition term 정의 용어 용어의 제목을 넣을 때 사용
dd definition description 정의 설명 용어를 설명하는데 사용
